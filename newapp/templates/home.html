Welcome to Correct Easy

Correct Easy is an Django appplication which uses Google's Vision API to analyze images.
Correct Easy makes it easier for teachers to calculate the marks of each question . The teacher simply needs to upload the key words of the answer along with the image of the students answer. We will evaluate the answer and predict an approximate marks that could be given to the students.

The steps to use Correct Easy

  • Upload student's answer key
  • Enter answer key
  • Correct Easy predicts the marks

Upload student's answer

CorrectEasy makes it easier for teachers to calculate the marks of each question. In the first step, the teacher simply needs to upload the answer sheet of the students.

Enter answer key

Once the teacher uploads the student answer sheet, next enter the answer key to identify the key points in the answer.

Mark prediction

Correct Easy helps evaluate the answer and predict the approximate mark that can be given to the students. With the help of Google's Vision API, the handwritten texts can be quickly identified and helps get the results faster

Features available in Correct Easy

Label Detection

Provides generalized labels for an image. For each label returns a textual description, confidence score, and topicality rating.

Text Recognition

Optical character recognition (OCR) for an image; text recognition and conversion to machine-coded text. Identifies and extracts UTF-8 text in an image.

Handwriting Recognition

Optimized for dense areas of text in an image (images that are documents), and images that contain handwriting.

Crop Hint Recognition

Image properties

Returns dominant colors in an image. Each color is represented in the RGBA color space, has a confidence score, and displays the fraction of pixels occupied by the color [0, 1].
